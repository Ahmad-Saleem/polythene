# Layout

<a class="btn-demo" href="http://arthurclemens.github.io/Polythene-examples/index.html#/layout">Demo</a>

Provides common and flexbox classes.

## Usage

```javascript
import { addLayoutStyles } from 'polythene-utilities';

addLayoutStyles();
```

Use in Mithril elements:

~~~javascript
m(".layout.vertical", ...)
~~~
